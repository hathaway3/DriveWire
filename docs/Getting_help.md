## Support

DriveWire 4 is a whole new code base written in a new language. We've done quite a bit of testing but there are certainly going to be issues that come up only as the software is tried by more people. 

You can often find other users of DW online in our IRC channel, #coco_chat on freenode. To join the room with your web browser, [click here](http://webchat.freenode.net/?channels=coco_chat).
