# Beta Testing

We welcome users interested in testing new features and reporting bugs in pre-release versions of DriveWire.

## 🛠️ How to Participate

1.  **Backup Your Data**: Always backup your virtual disks and configuration files before testing beta software.
2.  **Check Discord/Mailing List**: Beta announcements are typically made on the CoCo Mailing List or Discord.
3.  **Download Betas**: Beta builds are often available as separate branches or pre-releases on GitHub.

## 📝 Reporting Feedback

Please provide as much detail as possible when reporting issues:
- Host Operating System (Linux, Windows, macOS, etc.)
- DriveWire Implementation (C, Swift, MicroPython, etc.)
- Specific steps to reproduce the issue.
