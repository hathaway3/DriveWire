# DriveWire Server (Delphi)

This folder contains the DriveWire server implementation written in Delphi (Object Pascal). This version provides a native Windows solution with a focused set of visual components for managing virtual resources.

## 📋 Overview

The Delphi implementation allows DriveWire to be integrated into Windows-based development workflows.

Key components:
*   `DWServer.pas`: Main server application logic.
*   `VirtualDrive.pas`: Emulation logic for virtual disk drives.
*   `VirtualPrinter.pas`: Emulation logic for virtual printers.
*   `DWServer.dfm`: Visual form definitions for the server interface.

## 🔨 Build Instructions

1.  **IDE**: Open the project using Embarcadero RAD Studio (Delphi).
2.  **Dependencies**: Uses standard VCL components.
3.  **Compile**: Use the IDE's build commands to generate the `DWServer.exe` executable.

## ✅ Status

- **Target**: Windows 7, 10, and 11.
- **Stability**: Legacy. This implementation reflects the original Windows GUI tools.
