# DriveWire Host (Swift)

This folder houses the modern DriveWire host application implemented in Swift. It provides a native experience for macOS users, leveraging the language's safety and expressiveness to build a robust client-side component.

## 📋 Overview

The Swift implementation focuses on providing a user-friendly and extensible interface to the DriveWire server.

It includes:
*   `DriveWire.xcodeproj`: The Xcode project file for building the application.
*   `DriveWire`: The main Swift application code, responsible for UI and host logic.
*   `drivewire-cli`: A command-line interface (CLI) tool for interacting with the DriveWire environment.

## 🔨 Build Instructions

1.  **Prerequisites:** Requires a Mac with Xcode 12 or later.
2.  **Open Project:** Open `DriveWire.xcodeproj` in Xcode.
3.  **Build:** Select a build target (e.g., "DriveWire" or "drivewire-cli") and build the project (Cmd+B).

## ✅ Status

- **Status**: Active development.
- **Target**: macOS 11.0+.
- **Features**: Includes a modern UI and a growing CLI toolset.
